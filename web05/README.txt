This is a sample app like chatgpt using Ollama API and Llama as LLM.

pre-requisite:
1. Make sure the hardware being used has at least 8GB of RAM. Much better if it has dedicated NVIDIA GPU.
2. Install Ollama (https://ollama.ai/download) and do setup Llama2 as model(https://ollama.ai/library/llama2/tags).
3. Make sure Ollama is running in systemctl before executing the backend.
4. Make sure Flask installed.
