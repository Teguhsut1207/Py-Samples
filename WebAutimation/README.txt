This is a snippet of web automation project.

Story behind:
Client asked for a help to entry bunch of data in Excel into a website. We don't have direct access to the db nor server, so the only way is through the form in frontend.

Mandatory tech used:
- Python 3.11
- Selenium webdriver
- Pandas

how does it work:
- login to website
- load the data from excel into variables
- fill the form and submit
