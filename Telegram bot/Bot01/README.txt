This is a telegram bot connected with a LLM engine from Ollama. This sample using Llama2 model.

Things to do before use it:
1. download and setup Ollama (https://ollama.com/download/linux)
2. set Llama 2 as the model (https://ollama.com/library/llama2)
3. create a Telegram account
4. get a Telegram bot api token from botfather (https://core.telegram.org/bots/tutorial)
